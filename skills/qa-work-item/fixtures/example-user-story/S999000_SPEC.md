## Problem Statement

Provide a greeting file with the exact content `Hello, World!`. Used as a fixture target for `/qa-work-item`.

## Mental Model

```
fixture-impl.txt → must contain → "Hello, World!" (exact, capital W, with exclamation)
```

## Requirements

### P0 (Must-Have)

| # | Tag | What it asks | As a... | I want to... | So that... |
|---|-----|-------------|---------|-------------|------------|
| 1 | core | Does fixture-impl.txt contain the exact greeting? | /qa-work-item test target | verify content matches `Hello, World!` exactly | the subagent has a verifiable assertion |
| 2 | core | Is fixture-impl.txt present and non-empty? | /qa-work-item test target | confirm presence and non-emptiness via smoke | smoke runs cheaply before subagent dispatch |

## Acceptance Criteria

### Story #1: Exact content match [core]

```
GIVEN  fixture-impl.txt exists
WHEN   the file is read
THEN   its content equals "Hello, World!" (exact bytes, no surrounding whitespace beyond a single trailing newline)
```

### Story #2: File present and non-empty [core]

```
GIVEN  the fixture directory
WHEN   smoke checks run
THEN   fixture-impl.txt exists
       AND its size > 0 bytes
```

## Architecture

```
fixture-impl.txt
       │
       ▼
   smoke: existence + non-empty (cheap)
       │
       ▼
   E2E: exact-content match (subagent reads + diffs)
```

### Components Affected

| Component | Repo | Change Type | Description |
|-----------|------|------------|-------------|
| `skills/qa-work-item/fixtures/example-user-story/fixture-impl.txt` | claude-skills-templates | New | Planted-bug greeting file |

### Data Flow

1. /qa-work-item reads TEST-SPEC
2. Smoke runs: existence + non-empty checks pass
3. Subagent reads fixture-impl.txt; diffs against expected `Hello, World!`
4. Subagent reports red (planted bug); writes journal entry

## Tradeoffs

| Decision | Chosen | Rejected Alternative | Why |
|----------|--------|---------------------|-----|
| Bug placement | Content (string mismatch) | Structure (missing file) | Smoke would catch missing file; we want the bug visible only at E2E layer |
| Greeting choice | `Hello, World!` (capital W, !) vs `Hello, world` (lowercase, no !) | Numerical mismatch | String diffs are easy for the subagent to express in 1-2 sentences |
