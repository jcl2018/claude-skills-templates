## Overview

Take the SLOW inline doc-sync + test-sync off the cj_goal build path via a
deterministic-agentic split; keep the fast deterministic sync inline; defer the
slow agentic sync to the existing nightly audit; enforce the slimmed shape through
the two-axis test contract.

## Data Flow

Orchestrator QA dispatch (`DEFER_AUDIT: true` + `DEFER_SYNC: true`) → qa.md 8.6.0
sets both flags → 8.6a/8.6b run deterministic-only, skip the agentic sweep; 8.6c/8.6d
skipped → pre-doc-sync commit → Step 5.5 deterministic `--render-docs` regen (commits
docs/ delta) → `/ship`. Nightly: `audit-nightly.yml` → `/CJ_doc_audit` + `/CJ_test_audit`
→ `audit-drift` issue (the deferred-drift safety net).

## Out of Scope

nightly doc-sync PR — see the ROADMAP Non-Goals.
