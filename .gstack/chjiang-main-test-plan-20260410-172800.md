## Test Tiers (extend existing test.sh structure)

### Tier 5: Lifecycle Script Smoke Tests

| Test | Script | Assertion |
|------|--------|-----------|
| skill-design.sh creates DESIGN.md | skill-design.sh zzz-test-lifecycle | File exists at skills/zzz-test-lifecycle/DESIGN.md |
| skill-design.sh fails on existing | skill-design.sh zzz-test-lifecycle (2nd time) | Exit 1 |
| skill-design.sh rejects invalid name | skill-design.sh INVALID | Exit 1 |
| skill-design.sh rejects version-collision name | skill-design.sh foo-v2 | Exit 1 |
| create-skill.sh creates CHANGELOG.md | create-skill.sh zzz-test-lifecycle | File exists at skills/zzz-test-lifecycle/CHANGELOG.md |
| create-skill.sh adds version to frontmatter | create-skill.sh zzz-test-lifecycle | grep 'version:' in SKILL.md |
| skill-check.sh passes valid skill | skill-check.sh zzz-test-lifecycle | Exit 0 |
| skill-check.sh fails missing DESIGN.md | (remove DESIGN.md) | Exit 1 |
| skill-check.sh fails missing CHANGELOG | (remove CHANGELOG.md) | Exit 1 |
| skill-check.sh fails version mismatch | (edit frontmatter version) | Exit 1 |

### Tier 6: Version & Ship Tests

| Test | Script | Assertion |
|------|--------|-----------|
| skill-version.sh bumps patch | skill-version.sh zzz-test-lifecycle patch | version = 0.1.1 in frontmatter AND catalog |
| skill-version.sh bumps minor | skill-version.sh zzz-test-lifecycle minor | version = 0.2.0 |
| skill-version.sh bumps major | skill-version.sh zzz-test-lifecycle major | version = 1.0.0 |
| skill-version.sh rejects invalid type | skill-version.sh zzz-test-lifecycle foo | Exit 1 |
| skill-version.sh rejects nonexistent skill | skill-version.sh nonexistent patch | Exit 1 |
| skill-version.sh adds CHANGELOG header | After bump | grep '## [0.1.1]' in CHANGELOG.md |
| skill-version.sh rollback on failure | (make catalog read-only) | Frontmatter unchanged |
| skill-ship.sh fails on duplicate tag | (create tag, run again) | Exit 1 |
| skill-ship.sh fails with pre-existing staged changes | (stage unrelated file) | Exit 1 |

### Tier 8: Integration Tests

| Test | Script | Assertion |
|------|--------|-----------|

### Cleanup

All tests use `trap EXIT` to:
- Remove test skill directories
- Remove test catalog entries
- Remove test git tags (`git tag -d zzz-test-lifecycle-v*`)
- Restore modified files
